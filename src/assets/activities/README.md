# Activity Images

This directory contains images used for the Discover Panel activities.

## Image Specifications
- **Format**: `.jpg` or `.png`
- **Recommended Size**: 800x1200px (for `span: 2`) or 800x600px (for `span: 1`)
- **Naming Convention**: Matches the `id` in `src/constants/activities.ts`

## Files
- `chat.jpg`: Used for the "Zen Chat" activity.
- `mind-map.jpg`: Used for the "Mandala Map" activity.

Note: Images are referenced by path string in the code to allow for easy swapping without rebuilding.
